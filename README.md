# 言葉のリフレクションゲーム

Zoom画面共有を前提とした、子どもたちが「ことばを通して自分をふり返る」オンライン活動支援アプリです。

## 📝 概要

- **対象**：小中学生（オンライン支援・国語・SST・自己理解プログラムなど）
- **実施形態**：ゲームマスター（GM）がアプリをデスクトップで開き、Zoomで画面共有
- **回答方法**：子どもはZoomチャットで回答

## ✨ 機能

- **お題デッキ**：40問のお題（語感・言い換え・比喩・視点変換・抽象化・具体化・メタ認知・価値観）
- **カテゴリフィルタ**：お題をカテゴリで絞り込み
- **難易度フィルタ**：★1〜★3の難易度で絞り込み
- **シャッフル機能**：お題をランダムに並び替え
- **キーボード操作**：
  - `Space` / `→`：次のお題へ
  - `←`：前のお題へ
  - `S`：シャッフル
  - `F`：フルスクリーン切り替え
- **フルスクリーン対応**：Zoom共有時に見やすく

## 🚀 使い方

### ローカルで実行

```bash
# 依存パッケージをインストール
npm install

# 開発サーバーを起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

### デプロイ

mainブランチにpushすると、GitHub Actionsが自動的にビルド・デプロイします。

## 🎨 技術スタック

- **フロントエンド**：React + TypeScript
- **ビルドツール**：Vite
- **スタイリング**：TailwindCSS
- **デプロイ**：GitHub Pages
